# 🌱 Casulo Tecnológico vs 2.0

Bem-vindo(a) ao **Casulo Tecnológico** — um site interativo e informativo desenvolvido com **HTML, CSS e JavaScript**, projetado para apresentar essa iniciativa de formação de talentos em tecnologia.

🔗 **Demo Online:**  
https://casulo-tech.vercel.app/

---

## 🧠 Sobre o projeto

O *Casulo Tecnológico* é uma página web institucional que apresenta uma proposta de capacitação e desenvolvimento de profissionais em tecnologia.  
A estrutura foi construída com foco em layout moderno e interação fluida, utilizando apenas **tecnologias web básicas** (sem frameworks no front-end), com ênfase em animações CSS e manipulação simples com JavaScript.

Este projeto visa demonstrar habilidades em front-end puro, experiência de usuário e organização de conteúdo de maneira clara e atrativa.

---

## 📌 Funcionalidades

- Layout responsivo com HTML5 e CSS3
- Animações CSS para transições e efeitos visuais
- Interações com JavaScript (validação, controle de estados ou animações)
- Seções organizadas:
  - Início
  - Sobre Nós
  - Como Funciona
  - O que Oferecemos
  - Inscrições (formulário)
  - Contato

---

## 🛠️ Tecnologias utilizadas

Este projeto foi construído com:

- **HTML5**
- **CSS3** (incluindo animações com `@keyframes` e transições)
- **JavaScript (Vanilla JS)**
